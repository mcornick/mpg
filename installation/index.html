Installation

I use this repository to experiment with goreleaser, so there are a ridiculous number of builds and packages available for such a trivial app. Don't judge.

Homebrew

brew install mcornick/tap/mpg

Scoop

scoop bucket add mcornick https://github.com/mcornick/scoop-bucket.git
scoop install mpg

Docker

docker run --rm ghcr.io/mcornick/mpg

Signatures

Binary checksums and container manifests are signed with Cosign. The certificate needed to verify signatures can be found in each release.
